# GB18030 中文编码字符集

## 说明
GB18030是中华人民共和国《中文编码字符集》国家标准，最新版本为GB18030-2022。
本项目收录了GB18030-2022标准规范(PDF)、GB18030-2022编码到UCS编码转换表等文档，同时还有编程语言实现，可方便的处理、转换GB18030-2022字符串。
